# Rhodium - SOTA Tower Project Dashboard

## Getting Started

### Installation
```bash
npm install
```

### Development Server
```bash
npm run dev
```

This will start the development server at `http://localhost:3000` and automatically open your default browser.

### Features
- **Interactive Gantt Charts** - Detailed timelines for every component
- **Quality Checklists** - SOTA-level installation requirements
- **Real-time Updates** - Hot reload with no caching issues
- **Multi-Tab Navigation** - Separate views for each room/system

## Project Structure
- `index.html` - Main dashboard application
- `Project_Recovery_Plan.md` - Detailed project documentation
- `package.json` - Node.js dependencies
- `vite.config.js` - Development server configuration
